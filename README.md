# Array

Создан класс Массив.

Разработаны service-классы реализующие функциональности:
* поиск min\max значения массива,
* замену элементов массива по условию,
* определение среднего значения элементов массива,
* определение суммы элементов массива,
* определение числа положительных\отрицательных элементов массива.
* сортировка массива тремя различными алгоритмами (быстрая сортировка, сортировка слиянием, сортировка подсчетом).

Параметры, необходимые для создания массивов, получены чтением информации из файла (.txt). Часть данных должна некорректна. Если встретилась некорректная
строка, приложение переходит к следующей строке. Чтение прекращается по достижении первой корректной строки.

Использованы собственные классы исключительных ситуаций.

Разработан validation-класс для валидации исходных данных при создании массивов.

Вышеуказанные задания решены также с помощью возможностей IntStream.
